# Snowflake QBR — CIENA, SAGENT & US FINTECH

Account strategy and GTM tools for the Snowflake Enterprise Account Executive territory.

## Contents

- **GTM Command Center** — Single-page account dashboard to track activity, signals, and next actions across Ciena, Sagent, and U.S. Financial Technology (live at [snowflake-qbr.vercel.app](https://snowflake-qbr.vercel.app))
- **[AI Territory Assistant Setup](ai-territory-assistant-setup.md)** — Instructions for Custom GPT (ChatGPT) or Claude Project trained on this strategy
- **[AI Outreach Prompt Templates](ai-outreach-prompt-templates.md)** — Ready-to-use prompts for research, outreach, discovery, and competitive intel

## Quick Start

1. **[Live demo](https://snowflake-qbr.vercel.app/)** — Deployed on Vercel
2. Or run locally: `python3 -m http.server 8080` from the repo root (then open http://localhost:8080)
2. Follow `ai-territory-assistant-setup.md` to configure your AI assistant
3. Use the prompt templates for weekly research and outreach
